## Part 1: Template System (4 days)

### 8 Built-in Templates

1. **basic-network** (3 devices)
   - 1 Router, 2 Switches
   - LLDP, CDP

2. **small-office** (5 devices)
   - 1 Router, 2 Switches, 1 Server, 1 AP
   - LLDP, CDP, DHCP, DNS, HTTP

3. **data-center** (10 devices)
   - 2 Core Routers, 4 Distribution Switches, 4 Servers
   - LLDP, CDP, STP, SNMP, HTTP

4. **isp-core** (15 devices)
   - 5 Core Routers, 10 Edge Routers
   - All protocols

5. **campus-network** (20 devices)
   - 2 Core Switches, 6 Distribution, 12 Access
   - LLDP, CDP, STP, DHCP, DNS

6. **snmp-testing** (5 devices)
   - Focused on SNMP traps and monitoring
   - All trap types, thresholds

7. **discovery-lab** (4 devices)
   - LLDP, CDP, EDP, FDP
   - Multi-vendor discovery

8. **dhcp-lab** (3 devices)
   - DHCPv4 and DHCPv6 servers
   - Multiple pools, options

## Part 2: Template Commands (2 days)

```bash
niac templates list            # List all templates
niac templates show basic      # View template details
niac config create --template basic  # Create from template
niac templates create --from my.yaml  # Save custom template
```
